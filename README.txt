# Virtual-Lion-Game
This is my first Java game. It's a text game about a lion's survival accompanied by images.

AUTHOR: Sino Oulad Daoud
DATE: Oct. 5, 2017

INSTRUCTIONS: Download project and extract files from pet-images.zip. Open the project on an IDE and create a new VirtualLionM
object, passing in a "name" for the lion surrounded by quotations. Play with the methods that feed, sleep, play, etc. and
respond to the statements given by the program by running more methods.
